=== Project Requirements
As seen above, although all main features in our application have been implemented, there aren't many concrete unit tests available. The only main unit tests made so far for our most important requirements are GraphServiceTest.java, RedirectControllerTest.java, GraphDSTest.java and DependencyControllerTest.java.

The application can take input in the form of a zip folder successfully, can fetch all mods within it's dependencies with the backend implementation, and create a graph with it, meeting the core requirements, although lacking a few concrete unit tests.
