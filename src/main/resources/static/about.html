About This Project

This Minecraft Mod Analyzer is an application developped as part of the INSO4101 Introduction to Software Engineering course at the University of Puerto Rico - Mayagüez Campus

Users enjoy changing their vanilla minecraft experience by including mods or modpacks. These mods include dozens of dependencies and changes on the original game which could make installing certain mods together unstable. This could lead to crashes or malfunctioning mods.

Then the goal of this project is to build a tool that detect these possible incompatibilities or conflicts that can cause problems, and note them to the user. All done, just by uploading the mods/modpack files.
