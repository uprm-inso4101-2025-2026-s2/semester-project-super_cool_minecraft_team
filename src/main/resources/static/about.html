About This Project

Users enjoy changing their vanilla Minecraft experience by including mods or modpacks. These mods include dozens of dependencies and changes to the original game, which could make installing certain mods together unstable. Which could lead to crashes or malfunctioning mods.

So, the goal of this project is to build a tool that detects these possible incompatibilities, or conflicts, that can cause problems when ran in-game and identify them to the user. All done, just by uploading the mods/modpack files.

This project, the Minecraft Mod Analyzer tool, is a web application developed as part of the INSO4101 Introduction to Software Engineering course at the University of Puerto Rico - Mayagüez Campus.
