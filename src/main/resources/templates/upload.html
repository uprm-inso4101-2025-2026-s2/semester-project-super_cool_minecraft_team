Windows Instructions

How to Create a .zip File on Windows

Follow these steps to compress your Minecraft modpack folder into a .zip file:

  1. Open File Explorer.
  2. Locate the folder that contains your Minecraft modpack.
  3. Right-click the folder.
  4. Select Send to → Compressed (zipped) folder.
  5. A .zip file will be created which you can upload to the analyzer.

macOS Instructions

How to Create a .zip File on macOS

Follow these steps to compress your Minecraft modpack folder into a .zip file:

  1. Open Finder.
  2. Locate the folder that contains your Minecraft modpack.
  3. Right-click the folder.
  4. Select Compress.
  5. A .zip file will be created which you can upload to the analyzer.

Linux Instructions

How to Create a .zip File on Linux

Follow these steps to compress your Minecraft modpack folder into a .zip file:

  1. Open your file manager.
  2. Locate the folder that contains your Minecraft modpack.
  3. Right-click the folder.
  4. Select Compress or Create Archive.
  5. A .zip file will be created which you can upload to the analyzer.

Maximum file size: 1GB
