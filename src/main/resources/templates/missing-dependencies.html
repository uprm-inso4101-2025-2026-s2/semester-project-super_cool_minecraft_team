How to install

  1. Open the dependency link for each missing mod.
  2. Download the file that matches your Minecraft version.
  3. Make sure the mod also matches your loader (Fabric).
  4. Place the downloaded .jar file into your modpack’s mods folder.
  5. After installing the missing mods, re-run analysis.

Warnings

  • Do not install a Forge mod in a Fabric modpack.
  • Do not install a mod built for a different Minecraft version.
  • If multiple files are listed, choose the one matching your current setup.

What to do if unresolved

  • Search for the dependency manually on Modrinth or CurseForge.
  • Confirm the exact mod ID, Minecraft version, and loader.
  • If the dependency still cannot be found, remove the parent mod or replace it with a compatible alternative.
  • After making changes, re-run analysis to verify the dependency graph again.
